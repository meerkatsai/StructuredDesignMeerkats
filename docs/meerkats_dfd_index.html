How the documents relate to each other
The DFDs (L0–L3) show data movement — what flows where, at increasing granularity. The ER Diagram shows data structure — what fields exist and how tables relate. The Flowchart shows execution logic — what decisions happen in what order. The Attribution Decision Tree is the detail inside P5 of the DFDs. The Identity Flow is the detail inside P1–P2. The Layer Summary is the prose narrative tying everything together. The Vertical Architecture shows how the schema adapts without changing — which is the core defensibility argument.
